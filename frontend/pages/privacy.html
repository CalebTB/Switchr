Privacy Notice (Skeleton)

This is a basic placeholder Privacy Notice page for the Switchr marketplace.

1) What We Collect (Prototype)

  • Account info (e.g., username, email)
  • Listing content (titles, descriptions, photos)
  • Offer details (trade items, simulated cash values)

2) How We Use It

  • To display listings and user profiles
  • To support offers and transaction history (simulated)
  • To support basic admin review and reporting

3) No Real Payment Data

Switchr does not handle real payment data. Any “payment” features are simulated.

4) Data Sharing

This is a student/prototype project. Data is not intended for commercial use and may be removed at any time.

5) Security

In a real system, passwords would be securely stored and user data would be protected. This prototype is for demonstration only.
